

- https://raw.githubusercontent.com/ckongari/scalata-image/main/images/dashboard/dashboard-view.png : Dashboard overview screen.

- https://raw.githubusercontent.com/ckongari/scalata-image/main/images/dashboard/click-to-access-chat-interface.png : How to open the chat interface from the Dashboard.
  - **What to click**: The **bottom chat input bar** (highlighted) with the placeholder text “What can I help you with today?” and a mic icon on the right.
  - **Location**: **Bottom-center** of the dashboard screen, spanning horizontally near the footer.
  - **Result**: Opens the **chat interface**.

- https://raw.githubusercontent.com/ckongari/scalata-image/main/images/news/click-on-create-newsfeed-quick-action-under-chat-input.png : Click the quick action to start creating a Newsfeed.
  - **What to click**: The **“Create Newsfeed” quick action button** shown just under the main chat input.
  - **Location**: **Below the chat input field** on the chat interface screen.
  - **Result**: Kicks the **Create Newsfeed** flow.

- https://raw.githubusercontent.com/ckongari/scalata-image/main/images/news/select-news-api-after-that-click-on-attach-api.png : Select a News API and attach it.
  - **What to click**: First select a **News API** from the list, then click the **“Attach API”** button.
  - **Location**: **Left side attachment drawer** under the **External APIs** tab.
  - **Result**: Attaches the chosen **News API** so the widget can pull news data from that source.

- https://raw.githubusercontent.com/ckongari/scalata-image/main/images/news/type-your-news-query-highlighing-text-input.png : Type your news query into the prompt field.
  - **What to click**: The **prompt text input** labeled “Enter a prompt here”.
  - **Location**: **Bottom-center chat input bar** in the chat interface.
  - **Result**: Lets the user **enter a query** that will be used to generate news content.

- https://raw.githubusercontent.com/ckongari/scalata-image/main/images/news/click-on-generate-newsfeed-widget-for-creating-newsfeed-widget.png : Generate the Newsfeed widget on the dashboard.
  - **What to click**: The **“Generate Newsfeed Widget” button** highlighted below the Latest News section.
  - **Location**: **Bottom-left of the Latest News card** area on the Chat response.
  - **Result**: Creates the **Newsfeed widget** on the dashboard.
